## 安裝方法
1. 檔案解壓縮之後會有兩個檔案，把他丟到有 NecroBot.exe 外掛那邊
2. 確保你的 Config/auth.json 和 Config/config.json 兩個檔案已經生成
3. 開啟 NecroBot Config GUI.exe，開始設定！
